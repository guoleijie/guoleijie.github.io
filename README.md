# 郭磊劼的个人主页 👋

> 简洁、现代、安全的个人主页，部署在 GitHub Pages

## 🎨 特点

- ✨ 现代深色主题设计
- 📱 响应式布局
- 🎭 打字机动画效果
- 📊 数字滚动动画
- ⬆️ 返回顶部按钮
- 🔒 安全优先（不暴露敏感信息）
- ⚡ 纯静态（无需后端）
- 🎯 简洁专业

## 📂 文件结构

```
personal-homepage/
├── index.html      # 主页
├── style.css       # 样式
├── script.js       # 交互脚本
├── README.md       # 说明文档
└── .gitignore      # Git 忽略文件
```

## 🚀 本地预览

```bash
# 进入项目目录
cd ~/.openclaw/workspace/personal-homepage

# 用浏览器打开
open index.html  # macOS
xdg-open index.html  # Linux
start index.html  # Windows
```

## 📦 部署到 GitHub Pages

```bash
# 1. 初始化 Git
git init
git add .
git commit -m "Initial commit"

# 2. 推送到 GitHub
git remote add origin https://github.com/guoleijie/guoleijie.github.io.git
git push -u origin main

# 3. 访问 https://guoleijie.github.io
```

## ✏️ 自定义内容

修改 `index.html` 中的以下部分：

- 姓名、职业标签
- 个人简介
- 技能卡片
- 项目展示
- 联系方式

## 🎨 自定义样式

修改 `style.css` 中的 CSS 变量：

```css
:root {
    --bg-color: #0a0a0a;           /* 背景色 */
    --text-primary: #ffffff;        /* 主文本色 */
    --accent-color: #6366f1;        /* 强调色 */
}
```

## 🔒 安全提醒

**✅ 推荐做法：**
- 使用 GitHub Issues 作为联系方式
- 只展示公开信息
- 定期检查页面内容

**❌ 避免暴露：**
- 真实邮箱（防爬虫）
- 电话号码
- 具体住址
- 身份证号
